Git repositoy address : https://github.com/hyunseok94/quadruped_robot.git

Package Name : Quadruped Robot
Sub Package : 
 (1) Quadruped Robot Description
     : This is for uploading model into the parameter, 'Robot description'.

 (2) Quadruped Robot Gazebo
     : This is for bringing up the Robot model to Gazebo World.

-----------------------------------------------------------------------------

Command for Run
(1) roscore
(2) cd catkin_ws/src && roslaunch quadruped_robot_gazebo bringup.launch (qb)

-------------------------------------------------------------------------------

Command for build
(1) cd ~/catkin_ws && catkin build (cm)

-------------------------------------------------------------------------------

Command for Edit
(1) cd ~/catkin_ws/src/quadruped_robot/quadruped_robot_gazebo/src && subl(ed)



-------------------------------------------------------------------------------
Commnad for plot
(1) cd ~/catkin_ws/src && rosrun plotjuggler plotjuggler (plot)
